Building Block: REACT with GraphQL

This Building Block consists of a GraphQL Apollo server (with express-graphql)
and a REACT Front-End with Apollo/client. Simple Create, Display and Delete operations
are made on a mock DB.

Summary:
- Back-End Nodejs GraphQl Apollo/server port 5000
- Front-End REACT GraphQl Apollo/client port 3000
- Bootstrap for minimum CSS
- Typescript at Frond-End
